# portfolio_asukaooi
1. プロジェクト概要
自販機シミュレーターの説明、消費者・業者の2ロール構成、あえてフレームワークを使わずServlet/JSPで実装した理由。
2. 技術スタック
Java17、Servlet、JSP、JSTL、Tomcat10、セッション管理。
3. 画面構成と機能一覧
消費者側（投入・購入・釣り銭）と業者側（補充・カスタム・売上回収）の機能を箇条書きで。
4. アーキテクチャの説明
MVCパターンとService層を追加した3層構造（Servlet→Service→DAO）を図か文章で説明。「なぜServiceを分けたか」も書く。
5. 工夫した点・こだわり
PRGパターンで二重送信防止
セッションスコープとリクエストスコープの使い分け
サーバー側で投入金額を管理（クライアント改ざん防止）
防御的プログラミング（nullチェック、例外の種類の使い分け）
6. 起動方法
EclipseとTomcatの設定手順、URL。
